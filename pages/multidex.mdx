## Description

Users can add an unlimited number of Modules on a single page by clicking the yellow "Add Module" button on the top of the page. 

Each module can have different DEXes or tokens selected, enabling users to view their entire portfolio on one screen at the same time.

Below you can see 4 modules open on 4 different DEXes - Uniswap, SushiSwap, PancakeSwap, & ShibaSwap
