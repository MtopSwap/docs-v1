## Modules

MtopSwap standardized the look and feel of hundreds of DEXes into one easy-to-understand interface (shown above), called a "Module".  This custom-built front-end has just a fraction of the code compared to other DEXes and is optimized for speed & end-user experience.

## Features

Each Module has all the features users are accustomed to:

- Dropdown list of DEXes on current ecosystem
- Token selection & import
- Token amount allocation (25%, 50%, 75%, max)
- Slippage customization
- Price impact
- Multihop toggle
- Transaction detail information

<Callout type="info" emoji="">
Selecting "max" token allocation will leave an appropriate amount of native tokens in a user's wallet to avoid insufficient gas funds for future transactions.
</Callout>
