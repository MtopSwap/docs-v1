<ImageWithModal src={pricecharts} alt="MultiDEX Showcase"/>

## Description

Once a token pair has been selected on a Module, the red "Chart" tab will light up.

Clicking it will bring up the price chart, with full functionality offered on DexScreener, such as changing timeframe, drawing lines, adding indicators, etc. 

There is no limit to the number of price charts users can open simultaneously.
